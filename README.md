
# SouthParkSolutions Reservation System

Este repositorio contiene la solución propuesta por SouthParkSolutions para un sistema de reservación de espacios de trabajo. El objetivo del sistema es proporcionar una plataforma sencilla e intuitiva que permita a los empleados consultar y reservar espacios disponibles dentro de la empresa.

El sistema ofrece una interfaz web donde los usuarios pueden visualizar la disponibilidad de la oficina, salas de trabajo individuales, salas de reunión u otros espacios de trabajo en tiempo real. A través de esta plataforma, los empleados pueden realizar reservas, verificar horarios disponibles y gestionar el uso de los espacios compartidos de forma eficiente.


## Funcionalidad

- Consultar disponibilidad en tiempo real
- Reservar espacios de trabajo
- Reservar salas de juntas
- Gestionar reservas de empleados
- Siendo manager uno puede consultar las analiticas de asistencia de su equipo


## Tech Stack

**Client:** React

**Server:** NodeJS, MySQL

**Sistema de sugerencias:** OpenAI API
